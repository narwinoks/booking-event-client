## Features

- Responsive web
- Login And Register User
- Home Page 
- Filter Event 
- User Profile
- Order Ticket 
- Checkout
- Update Profile 
- Other
- Admin Dashboard (comming son)


## Run Locally

Clone the project

```bash
  git clone https://github.com/narwinoks/MERN-Stack-E-commerce-Client/
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Open Server.js

```bash
  export const server = "<yOUR DOMAIN>/api/V1";
```

Start the server

```bash
  npm run start
```
